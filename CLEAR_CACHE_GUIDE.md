# 清除缓存和重置角色指南

## 问题描述
Aoba 角色在对话中说"我可不是Aoba"，说明角色配置没有正确加载。

## 解决方案

### 1. 清除浏览器缓存
1. 打开浏览器开发者工具 (F12)
2. 右键点击刷新按钮
3. 选择"清空缓存并硬性重新加载"
4. 或者按 Ctrl+Shift+R (Windows) / Cmd+Shift+R (Mac)

### 2. 清除本地存储
1. 打开浏览器开发者工具 (F12)
2. 进入 Application/应用程序 标签
3. 在左侧找到 Local Storage
4. 删除所有 `kuro-` 开头的项目

### 3. 重启服务器
```bash
# 停止当前服务器 (Ctrl+C)
# 然后重新启动
cd vue-kuro/server
node server.js
```

### 4. 清除聊天记录
1. 在聊天界面点击垃圾桶图标
2. 确认清除所有聊天记录
3. 重新开始对话

### 5. 验证角色配置
访问以下URL确认角色配置正确：
- http://localhost:3002/api/characters/koru

应该返回：
```json
{
  "id": "koru",
  "name": "Aoba",
  "avatar": "/img/Aoba(normal).png",
  ...
}
```

## 测试步骤
1. 清除缓存后重新访问应用
2. 选择 Aoba 角色
3. 发送消息："你好，Aoba"
4. 确认回复中角色自称是 "Aoba"

## 如果问题仍然存在
1. 检查浏览器控制台是否有错误
2. 确认服务器正在运行
3. 尝试使用不同的浏览器测试
